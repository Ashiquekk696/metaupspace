## Fonts

Drop your font files in this folder.

This app is currently configured to use the **Poppins** font family from:

- `assets/fonts/Poppins-Regular.ttf`
- `assets/fonts/Poppins-Medium.ttf`
- `assets/fonts/Poppins-SemiBold.ttf`
- `assets/fonts/Poppins-Bold.ttf`

If these files are missing, Flutter will throw an asset error at runtime.

### Suggested source

Download “Poppins” from Google Fonts and place the `.ttf` files here (matching the
exact names above).
